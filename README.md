Student Performance Dashboard
Project Overview

This project is built using Power BI to analyze student performance.

 Objectives

* Analyze student marks and attendance
* Identify top performing students
* Compare performance by subject and class

 Tools Used

* Power BI
* Excel

 Dashboard Features

* KPI Cards (Avg Marks, Attendance, Total Students)
* Subject-wise analysis
* Gender comparison
* Attendance vs Marks analysis
* Student ranking
